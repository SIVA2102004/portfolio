# Portfolio - Jakka Siva Subramanyam Guptha

Professional portfolio website showcasing projects, experience, and certifications.

## Features
- Interactive Bento Grid layout
- AI Builder & Full-Stack Developer focus
- Project showcase with GitHub links
- Certification gallery
- Contact form integration
- Dark/Light mode toggle
- Responsive design

## Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript
- **Styling:** Vanilla CSS (Modern CSS properties)
- **Icons:** Font Awesome
- **Fonts:** Outfit (Google Fonts)

## Project Structure
- `index.html`: Main landing page
- `gallery.html`: Photo gallery
- `certificates.html`: Certification listing
- `style.css`: Primary styling
- `bento.css`: Bento grid specific styles
- `certs.css`: Certification specific styles
- `script.js`: Interactive functionality

## Usage
Open `index.html` in any modern web browser.
